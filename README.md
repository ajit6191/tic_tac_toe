<h1 align="center">TIC-TAC-TOE-GAME</h1>

## :rocket: Knowledges

- `ReactJS`

## Clone this repository

```
git clone https://github.com/ajit6191/tic_tac_toe.git
```

## Go into the repository

```
cd tic_tac_toe
```

## Install dependencies

```
npm install // pnpm
```

## Run the app

```
npm run dev
```
